# KAGUJJE Portfolio \u2014 *just a brand*\n\nA professional, dark-themed portfolio website for the **KAGUJJE Brand** with a full-featured admin dashboard. Built with **Next.js 15**, **Tailwind CSS**, and **Supabase** (Auth, Database, Storage).\n\n\ud83c\udf10 **Live domain**: [www.kagujje.com](https://www.kagujje.com)\n\n---\n\n## \u2728 Features\n\n### Landing Page\n- Elegant dark theme with gold accent color\n- Hero section with profile picture, brand motto, and social links\n- **Business Units** section \u2014 showcase sub-brands and ventures\n- Projects gallery with tags, images, and live/repo links\n- Writings/blog section with full article pages\n- Contact section with email and location\n- Fully customizable \u2014 all headings and text editable from admin\n\n### Admin Dashboard (`/admin`)\n- **Secure login** via Supabase Auth\n- **Profile Management** \u2014 avatar, name, tagline, bio, resume\n- **Projects** \u2014 CRUD with image upload, tags, featured flag\n- **Writings** \u2014 CRUD with cover image, publish/draft\n- **Business Units** \u2014 manage sub-brands/ventures\n- **Links** \u2014 social/external links manager\n- **Page Sections** \u2014 edit every text section on landing page\n- **Files** \u2014 upload/manage files via Supabase Storage\n- **Settings** \u2014 site title, motto, accent color, SEO\n\n---\n\n## \ud83d\ude80 Setup Guide\n\n### 1. Create a Supabase Project\n\n1. Go to [supabase.com](https://supabase.com) and create a new project.\n2. Note your **Project URL** and **Anon Key**.\n\n### 2. Set Up the Database\n\n1. Go to the **SQL Editor** in your Supabase dashboard.\n2. Run `supabase/schema.sql` and `supabase/storage.sql`.\n\n### 3. Create an Admin User\n\n1. Go to Supabase Dashboard \u2192 Authentication \u2192 Users\n2. Click **Add User** with email/password\n3. Use these credentials to log into `/admin`\n\n### 4. Environment Variables\n\nCreate `.env.local`:\n\n```env\nNEXT_PUBLIC_SUPABASE_URL=https://your-project-id.supabase.co\nNEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here\nNEXT_PUBLIC_SITE_URL=https://www.kagujje.com\n```\n\n### 5. Install & Run\n\n```bash\nnpm install\nnpm run dev\n```\n\n### 6. Deploy\n\nRecommended: **Vercel**\n```bash\nnpm i -g vercel\nvercel\n```\n\n---\n\n## \ud83d\udcc4 License\n\nPrivate \u2014 KAGUJJE Brand.\n
